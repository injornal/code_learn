# CodeLearn 

## How to run:
1. Build the frontend
  ```bash
  cd frontend/app
  npm run build
  ```
3. Install the requirements
  ```bash
  cd ../backend
  python3 -m venv .venv
  source .venv/bin/activate
  pip install -r requirements.txt
  ```
4. Run the server
  ```bash
  python main.py
  ```
5. The app should run at localhost:8080
